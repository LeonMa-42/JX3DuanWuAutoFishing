# JX3DuanWuAutoFishing

本工具用于剑网3端午钓鱼活动,支持原地复活自动进入钓鱼状态。

使用前需

## 使用方法

1. 在config.ini中设置出杆技能坐标、收杆技能坐标、上钩提示点坐标、上钩提示点颜色、HP检测点坐标、HP检测点颜色、原地复活按钮点坐标、鱼篓坐标。

2. 将视角拉远，角色头部大概与鱼篓重合点击鱼篓进入钓鱼状态，右键钓鱼工具以管理员身份运行。
